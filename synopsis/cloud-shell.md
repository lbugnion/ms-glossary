## Short description

Azure Cloud Shell is a command-line interface available via multiple access points, including the Azure portal, shell.azure.com, Azure mobile app, Azure docs (e.g. Azure CLI, Azure PowerShell ), and VS Code Azure Account extension. An interactive, authenticated, browser-accessible shell for managing Azure resources. Azure Cloud Shell provides the flexibility of choosing the shell experience that best suits the way you work, either Bash or PowerShell.
